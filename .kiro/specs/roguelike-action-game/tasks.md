# 实现计划：肉鸽动作游戏（Roguelike Action Game）

## 概述

按照依赖关系从底层到上层逐步实现：框架层新增模块 → 引擎层新增模块 → Excel 导出工具 → 游戏模块基础文件 → 数据接口层 → 运行时核心系统 → 可扩展类型实现 → UI 面板 → 战斗流程集成。所有代码使用 TypeScript 编写。
