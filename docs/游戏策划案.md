# 小游戏策划案

这是一个肉鸽游戏

初始1只【凡人】级棋子，打怪掉钱，用钱升级和刷新棋子。

棋子分为职业者和宝物等类型
棋子进化路线：
修仙系(成长缓慢，需要资源极多，但是战力惊人）：【凡人】->【练气】->【筑基】->【结丹】->【元婴】->【化神】->【炼虚】->【合体】->【渡劫】->【大乘】->【天线】->【真仙】->【玄仙】->【金仙】->【太乙】->【大罗】->【混元】
超凡系（成长快速，需要资源适中，战力中等）：【凡人】->【觉醒】->【精英】->【大师】->【史诗】->【传奇】
武道系（成长速度中等，需要资源适中，战力中等）：【凡人】->【武者】->【武师】->【武宗】->【武王】->【武圣】->【武尊】->【武皇】->【武帝】->【武神】

初始等级1级，给1个【凡人】棋子，战斗区域是宽高为15*30的2D网格，每个棋子占1格，每级可以多放1个棋子，3个相同棋子升级到下一级别，第一次升级时可以为棋子选择成长体系

战斗方式采用卡片魔王的模式
